# 关键路径计算说明（模板）

目的：对 WBS 任务依赖进行关键路径（CPM）分析，输出关键路径、里程碑与总工期。  

## 1. 输入
- `appendix/plan/WBS.csv`：包含任务 ID、工期、依赖  

## 2. 方法
1) 建立依赖图（DAG）：边为“依赖 → 被依赖”  
2) 前向计算：得到每个任务的最早开始 ES、最早完成 EF  
3) 后向计算：得到最晚开始 LS、最晚完成 LF  
4) 总时差 TF = LS - ES  
5) TF=0 的任务构成关键路径

- 关键路径：TBD（按任务 ID 串联）  
- 关键里程碑：M1..M6（与第三章一致）
