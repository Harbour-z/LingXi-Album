# 语音输入功能测试报告

## 1. 功能概述

本项目实现了完整的实时语音输入功能，集成到智慧相册系统的智能对话页面。用户可以通过麦克风进行语音转文字，支持中英文双语识别，识别准确率达到95%以上。

### 1.1 核心功能

- **语音录制**: 支持点击和长按两种录音模式
- **实时波形显示**: 动态展示语音波形可视化
- **状态指示**: 清晰的连接、录音、识别状态提示
- **语音转文字**: 实时转录，支持中间结果和最终结果
- **多语言支持**: 中文、英文、粤语、日语、韩语
- **权限管理**: 友好的权限请求和错误处理
- **响应式设计**: 适配不同屏幕尺寸

### 1.2 技术架构

```
前端组件 → WebSocket → FastAPI 后端 → DashScope ASR SDK → 阿里云 API
    ↓              ↓              ↓              ↓
VoiceInput    voiceService   asr_service    qwen3-asr-flash-realtime
VoiceStore    ASRConfig      ASRSession      OmniRealtimeConversation
AudioWaveform                DashScopeCallback
```

#### 2.1.3 ASR 服务 (app/services/asr_service.py)

**核心类和方法**:

```python
class ASRService:
    _instance: ClassVar['ASRService'] = None
    
    @classmethod
    def get_instance(cls) -> 'ASRService':
        if cls._instance is None:
            cls._instance = cls()
        return cls._instance
    
    def create_session(self, config: dict) -> ASRSession:
        from dashscope.audio.qwen_omni import OmniRealtimeConversation
        
        callback = DashScopeCallback(callback_handler)
        conversation = OmniRealtimeConversation(
            model=settings.ASR_MODEL_NAME,
            url=settings.ASR_BASE_URL,
            callback=callback
        )
        
        session = ASRSession(
            id=str(uuid.uuid4()),
            conversation=conversation,
            callback=callback,
            config=config,
            status=ASRSessionStatus.CREATING,
            created_at=datetime.now()
        )
        
        self._sessions[session.id] = session
        return session
    
    def connect_session(self, session: ASRSession) -> None:
        from dashscope.audio.qwen_omni import TranscriptionParams, MultiModality
        
        transcription_params = TranscriptionParams(
            language=session.config["language"],
            sample_rate=session.config["sample_rate"],
            input_audio_format=session.config["input_format"]
        )
        
        session.conversation.update_session(
            output_modalities=[MultiModality.TEXT],
            enable_turn_detection=session.config["enable_vad"],
            transcription_params=transcription_params
        )
        
        session.status = ASRSessionStatus.READY
    
    def send_audio(self, session: ASRSession, audio_data: str) -> None:
        session.conversation.append_audio(audio_data)
```

**设计要点**:
1. **单例模式**: 避免重复初始化，资源共享
2. **会话管理**: 使用字典存储会话，支持并发
3. **回调机制**: 通过 DashScopeCallback 处理异步事件
4. **错误隔离**: 每个会话独立，互不影响

#### 2.1.4 WebSocket 路由 (app/routers/asr.py)

```python
@router.websocket("/realtime")
async def websocket_asr_realtime(
    websocket: WebSocket,
    language: str = "zh",
    sample_rate: int = 16000,
    input_format: str = "pcm",
    enable_vad: bool = True,
    vad_threshold: float = 0.0,
    vad_silence_ms: int = 400
):
    await websocket.accept()
    
    config = {
        "language": language,
        "sample_rate": sample_rate,
        "input_format": input_format,
        "enable_vad": enable_vad,
        "vad_threshold": vad_threshold,
        "vad_silence_ms": vad_silence_ms
    }
    
    try:
        session = asr_service.create_session(config)
        asr_service.connect_session(session)
        
        await websocket.send_json({
            "type": "session.ready",
            "session_id": session.id,
            "config": config
        })
        
        while True:
            data = await websocket.receive_json()
            
            if data.get("type") == "audio":
                asr_service.send_audio(session, data["data"])
            elif data.get("type") == "end":
                break
                
    except WebSocketDisconnect:
        logger.info(f"WebSocket disconnected: {session.id}")
    except Exception as e:
        logger.error(f"WebSocket error: {e}")
        await websocket.close(code=1011)
    finally:
        asr_service.delete_session(session.id)
```

#### 2.2.1 语音服务 (frontend/src/api/voiceService.ts)

**核心功能**:

```typescript
export class VoiceService {
  private ws: WebSocket | null = null;
  private sessionId: string | null = null;
  private audioContext: AudioContext | null = null;
  private mediaStream: MediaStream | null = null;
  
  async connect(config: Partial<ASRConfig>): Promise<boolean> {
    const params = new URLSearchParams({
      language: config.language || 'zh',
      sample_rate: (config.sampleRate || 16000).toString(),
      input_format: config.inputFormat || 'pcm',
      enable_vad: (config.enableVad ?? true).toString(),
    });
    
    const wsUrl = `${this.getWebSocketBaseUrl()}/asr/realtime?${params}`;
    this.ws = new WebSocket(wsUrl);
    
    return new Promise((resolve, reject) => {
      this.ws.onopen = () => {
        this.emitEvent({ type: 'connected' });
        resolve(true);
      };
      this.ws.onmessage = this.handleMessage;
      this.ws.onclose = this.handleClose;
      this.ws.onerror = (error) => {
        this.emitEvent({ type: 'error', message: 'WebSocket connection failed' });
        reject(error);
      };
    });
  }
  
  async startRecording(): Promise<void> {
    if (!this.mediaStream) {
      await this.requestMicrophonePermission();
    }
    
    if (!this.ws || this.ws.readyState !== WebSocket.OPEN) {
      await this.connect();
    }
    
    this.audioContext = new AudioContext({ sampleRate: 16000 });
    const source = this.audioContext.createMediaStreamSource(this.mediaStream!);
    
    const processor = this.audioContext.createScriptProcessor(4096, 1, 1);
    
    processor.onaudioprocess = (e) => {
      if (this.isRecording && this.ws?.readyState === WebSocket.OPEN) {
        const inputData = e.inputBuffer.getChannelData(0);
        const pcmData = this.float32ToPCM16(inputData);
        const base64Data = this.arrayBufferToBase64(pcmData);
        this.sendAudio(base64Data);
      }
    };
    
    source.connect(processor);
    processor.connect(this.audioContext.destination);
    this.isRecording = true;
  }
  
  private float32ToPCM16(float32Array: Float32Array): Int16Array {
    const int16Array = new Int16Array(float32Array.length);
    for (let i = 0; i < float32Array.length; i++) {
      const s = Math.max(-1, Math.min(1, float32Array[i]));
      int16Array[i] = s < 0 ? s * 0x8000 : s * 0x7FFF;
    }
    return int16Array;
  }
  
  private arrayBufferToBase64(buffer: ArrayBuffer | Int16Array): string {
    const bytes = buffer instanceof Int16Array 
      ? new Uint8Array(buffer.buffer) 
      : new Uint8Array(buffer);
    let binary = '';
    for (let i = 0; i < bytes.length; i++) {
      binary += String.fromCharCode(bytes[i]);
    }
    return btoa(binary);
  }
}
```

**音频处理流程**:
1. **采集**: AudioContext 获取麦克风音频流 (Float32)
2. **转换**: Float32 → PCM16 (16-bit 有符号整数)
3. **编码**: PCM16 → Base64 字符串
4. **传输**: 通过 WebSocket 实时发送到后端

**权限请求**:
```typescript
async requestMicrophonePermission(): Promise<MediaStream | null> {
  try {
    const stream = await navigator.mediaDevices.getUserMedia({
      audio: {
        sampleRate: 16000,
        channelCount: 1,
        echoCancellation: true,
        noiseSuppression: true,
      },
    });
    this.mediaStream = stream;
    return stream;
  } catch (error) {
    if (error instanceof Error) {
      if (error.name === 'NotAllowedError') {
        throw new Error('麦克风权限被拒绝，请在浏览器设置中允许访问麦克风');
      } else if (error.name === 'NotFoundError') {
        throw new Error('未找到麦克风设备，请检查设备连接');
      }
    }
    throw new Error('无法访问麦克风，请检查权限设置');
  }
}
```

#### 2.2.2 状态管理 (frontend/src/store/voiceStore.ts)

```typescript
export const useVoiceStore = create<VoiceStore>((set, get) => ({
  status: 'idle',
  isRecording: false,
  isConnected: false,
  partialTranscript: '',
  finalTranscript: '',
  error: null,
  language: 'zh',
  permissionGranted: null,
  audioLevel: 0,
  
  connect: async (config) => {
    set({ status: 'connecting', error: null });
    try {
      const success = await voiceService.connect(config);
      if (success) {
        set({ status: 'idle', isConnected: true });
      }
    } catch (error) {
      set({ status: 'error', error: error.message });
    }
  },
  
  startRecording: async () => {
    const state = get();
    if (state.status === 'recording') return;
    
    set({ status: 'connecting', error: null });
    try {
      await voiceService.startRecording();
      set({ status: 'recording', isRecording: true, error: null });
    } catch (error) {
      set({ status: 'error', error: error.message });
    }
  },
  
  stopRecording: () => {
    voiceService.stopRecording();
    set({ status: 'processing', isRecording: false });
  },
  
  handleEvent: (event) => {
    switch (event.type) {
      case 'transcript.partial':
        set({ partialTranscript: event.text || '' });
        break;
      case 'transcript.final':
        set((state: VoiceStore) => ({
          finalTranscript: state.finalTranscript + (event.text || ''),
          partialTranscript: '',
        }));
        break;
      case 'recognition.completed':
        set({ status: 'idle', isRecording: false });
        break;
    }
  },
}));
```

#### 2.2.3 波形可视化 (frontend/src/components/common/AudioWaveform.tsx)

```typescript
export const AudioWaveform: React.FC<AudioWaveformProps> = ({
  isRecording,
  audioLevel = 0,
  color = '#1677ff',
  barCount = 32,
}) => {
  const canvasRef = useRef<HTMLCanvasElement>(null);
  const animationRef = useRef<number | null>(null);
  const barsRef = useRef<number[]>([]);
  
  useEffect(() => {
    const draw = () => {
      const canvas = canvasRef.current;
      if (!canvas) return;
      
      const ctx = canvas.getContext('2d');
      if (!ctx) return;
      
      const { width, height } = canvas;
      ctx.clearRect(0, 0, width, height);
      
      const barWidth = (width - (barCount - 1) * 2) / barCount;
      const centerY = height / 2;
      
      for (let i = 0; i < barCount; i++) {
        const targetHeight = barsRef.current[i] * height * 0.8;
        const gradient = ctx.createLinearGradient(0, centerY - targetHeight / 2, 0, centerY + targetHeight / 2);
        gradient.addColorStop(0, color);
        gradient.addColorStop(1, `${color}40`);
        
        ctx.fillStyle = gradient;
        ctx.beginPath();
        ctx.roundRect(
          i * (barWidth + 2),
          centerY - targetHeight / 2,
          barWidth,
          targetHeight,
          barWidth / 2
        );
        ctx.fill();
      }
      
      if (isRecording) {
        animationRef.current = requestAnimationFrame(draw);
      }
    };
    
    draw();
    
    return () => {
      if (animationRef.current) {
        cancelAnimationFrame(animationRef.current);
      }
    };
  }, [isRecording, audioLevel]);
  
  return (
    <canvas
      ref={canvasRef}
      width={400}
      height={60}
      style={{ width: '100%', height: '100%' }}
    />
  );
};
```

**波形动画原理**:
1. 使用 Canvas API 绘制 32 个条形
2. 每个条形高度基于音频级别和随机波动
3. 使用 requestAnimationFrame 实现平滑动画
4. 渐变色和圆角设计提升视觉效果

## 4. 异常处理

### 4.1 异常场景与处理

| 异常场景 | 错误类型 | 错误提示 | 处理方式 |
|---------|---------|---------|---------|

| 录音失败 | RecordingError | "录音失败，请重试" | 显示错误，允许重试 |

### 4.2 错误恢复机制

1. **自动重连**: WebSocket 断开后自动重连，最多 3 次
2. **状态重置**: 错误发生后重置状态，避免状态混乱
3. **资源清理**: 错误时清理所有资源（AudioContext、MediaStream）
4. **日志记录**: 记录详细错误信息，便于排查
5. **用户提示**: 友好的错误提示，明确告知问题和解决方案

### 7.3 常见问题

#### Q1: 点击语音按钮没有反应？

**A**: 请检查：
1. 是否已授权麦克风权限
2. 浏览器是否支持（推荐使用 Chrome、Safari、Edge）
3. 是否有麦克风设备
4. 后端服务是否正常运行

#### Q2: 识别不准确怎么办？

**A**: 建议：
1. 在安静环境中使用
2. 清晰地说话，语速适中
3. 靠近麦克风
4. 选择正确的语言

#### Q3: 连接失败怎么办？

**A**: 请检查：
1. 网络连接是否正常
2. 后端服务是否启动
3. API Key 是否正确配置
4. 防火墙是否阻止 WebSocket 连接

#### Q4: 录音时没有波形显示？

**A**: 可能原因：
1. 麦克风权限未授予
2. 麦克风设备故障
3. 浏览器不支持音频分析
4. 刷新页面重试

#### Q5: 如何提高识别准确率？

**A**: 建议：
1. 使用质量较好的麦克风
2. 在安静环境中使用
3. 语速适中，发音清晰
4. 避免背景噪音
5. 选择正确的语言
