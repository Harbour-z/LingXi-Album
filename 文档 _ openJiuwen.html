文档 / 4.开发指南 / openJiuwen Core开发指南 / 智能体 / 构建ReActAgent

本章节演示了如何基于openJiuwen构建一个用于天气查询的ReActAgent应用，该应用支持通过ReAct规划模式引导大模型生成插件调用命令，进而结合插件执行结果生成最终答案。通过示例，你将会了解到如下信息：

  • 如何创建提示词。
  • 如何使用插件模块。
  • 如何创建和执行ReActAgent。

应用设计流程

ReActAgent是一种遵循ReAct（Reasoning + Action）规划模式的Agent，通过 “思考（Thought）→ 行动（Action）→ 观察（Observe）”的循环迭代完成用户任务。

  1. 思考：ReActController调用LLM进行任务规划，解析 LLM 输出里是否包含工具执行指令。
  2. 行动：根据思考中 LLM 的输出，分两种情况进行操作：
    • 有工具执行指令：调工具，选择并调用合适的工具（如检索、数据库、第三方API、代码执行等）执行具体的操作，本用例中是调用一个根据时间和地点查询天气的工具；
    • 无工具执行指令：把LLM输出作为最终答案。
  3. 观察：ReActController把工具返回的Observation追加到对话历史，再调用LLM进行下一次任务规划。

ReActAgent能根据工具执行结果观察与反馈，不断调整策略，优化推理路径，直至达成任务目标或获得最终答案。其强大的多轮推理与自我修正能力，使ReActAgent具备动态决策能力且能够灵活应对环境变化，适用于需要复杂推理和策略调整的多样化任务场景。

前提条件

Python的版本应高于或者等于Python 3.11版本，建议使用3.11.4版本，使用前请检查Python版本信息。

安装openJiuwen

执行如下命令安装openJiuwen：

BASH
1pip install openjiuwen-x.y.z-py3-none-any.whl

创建提示词模板

创建系统提示词模板，用于设定ReActAgent的整体行为和角色定位。以下系统提示词不仅定义了人设、任务目标，同时提供了当前日期信息，还给定了约束限制，帮助ReActAgent在与用户交互时正确理解当前时间完成任务目标。示例代码如下：

PYTHON
1def build_current_date():
2    """获取当前日期"""
3    from datetime import datetime
4    current_datetime = datetime.now()
5    return current_datetime.strftime("%Y-%m-%d")
6

7def _create_prompt_template():
8    system_prompt = ("你是一个AI助手，在适当的时候调用合适的工具，帮助我完成任务！\n"  # 人设&任务目标
9                     "今天的日期为：{}\n"                                       # 当前日期
10                     "注意：1. 如果用户请求中未指定具体时间，则默认为今天。")         # 约束限制
11    return [
12        dict(role="system", content=system_prompt.format(build_current_date()))
13    ]

创建插件对象及其描述信息

本示例创建了天气查询插件，并定义了其输入参数的结构和要求。首先通过RestfulApi接口将天气查询服务封装成可以被框架使用的工具类。示例代码如下：

注意

注意
本地测试请求服务（http 开头），可以通过配置相关的环境变量关闭SSL校验。但禁用SSL会跳过证书验证，可能遭遇数据篡改、中间人攻击，导致敏感信息泄露，仅允许测试环境临时使用，生产环境务必启用SSL校验保障安全。

PYTHON
1import os
2from openjiuwen.core.utils.tool.service_api.restful_api import RestfulApi
3from openjiuwen.core.utils.tool.param import Param
4

5os.environ["LLM_SSL_VERIFY"] = "false"  # 关闭SSL校验仅用于本地调试，生产环境请务必打开
6os.environ["RESTFUL_SSL_VERIFY"] = "false"  # 关闭SSL校验仅用于本地调试，生产环境请务必打开
7

8def _create_tool():
9    weather_plugin = RestfulApi(
10        name="WeatherReporter",
11        description="天气查询插件",
12        params=[
13            Param(name="location", description="天气查询的地点，必须为英文", type="string", required=True),
14            Param(name="date", description="天气查询的时间，格式为YYYY-MM-DD", type="string", required=True),
15        ],
16        path="your weather search api url",  # 天气查询服务部署地址
17        headers={},
18        method="GET",
19        response=[],
20    )
21    return weather_plugin

同时通过PluginSchema接口创建了插件描述信息，该描述信息后续将成为大模型输入的一部分，引导大模型生成工具调用命令。示例代码如下：

PYTHON
1from openjiuwen.agent.common.schema import PluginSchema
2

3def _create_tool_schema():
4    tool_info = PluginSchema(
5        name='WeatherReporter',
6        description='天气查询插件',
7        inputs={
8            "type": "object",
9            "properties": {
10                "location": {
11                    "type": "string",
12                    "description": "天气查询的地点。\n注意：地点名称必须为英文",
13                    "required": True
14                },
15                "date": {
16                    "type": "string",
17                    "description": "天气查询的时间，格式为YYYY-MM-DD",
18                    "required": True
19                }
20            }
21        }
22    )
23    return tool_info

创建ReActAgent

首先使用openJiuwen提供的create_react_agent_config方法快速创建天气查询的ReActAgentConfig对象，涵盖ReActAgent相关的配置参数信息，如提示词定义及大模型配置信息等。示例代码如下：

PYTHON
1from openjiuwen.agent.react_agent import create_react_agent_config
2

3react_agent_config = create_react_agent_config(
4    agent_id="react_agent_123",
5    agent_version="0.0.1",
6    description="AI助手",
7    model=_create_model_config(),    # 大模型的配置信息
8    prompt_template=_create_prompt_template()  # 自定义提示词
9)

其中，_create_model_config用于定义大模型的相关配置信息，如模型提供商、模型名称、API调用和模型温度等信息：

PYTHON
1import os
2from openjiuwen.core.utils.llm.base import BaseModelInfo
3from openjiuwen.core.component.common.configs.model_config import ModelConfig
4

5API_BASE = os.getenv("API_BASE", "your api base")
6API_KEY = os.getenv("API_KEY", "your api key")
7MODEL_NAME = os.getenv("MODEL_NAME", "")
8MODEL_PROVIDER = os.getenv("MODEL_PROVIDER", "")
9

10def _create_model_config() -> ModelConfig:
11    return ModelConfig(
12        model_provider=MODEL_PROVIDER,
13        model_info=BaseModelInfo(
14            model=MODEL_NAME,
15            api_base=API_BASE,
16            api_key=API_KEY,
17            temperature=0.7,
18            top_p=0.9,
19            timeout=30,
20        ),
21    )

接着使用openJiuwen提供的ReActAgent类的构造函数实例化对象，包括天气查询助手配置，示例代码如下：

PYTHON
1from openjiuwen.agent.react_agent import ReActAgent
2

3react_agent = ReActAgent(react_agent_config)
4react_agent.add_tools([_create_tool()])

运行ReActAgent

创建完ReActAgent对象后，可以调用invoke方法，获取用户query的回复。ReActAgent的invoke方法实现了ReAct的规划流程，通过大模型生成计划，判断是否需要执行工具，如果需要，执行工具调用来完成任务，返回最终结果，否则直接返回模型输出结果，结束流程。示例代码如下：

PYTHON
1import asyncio
2

3result = asyncio.run(react_agent.invoke({"query": "查询杭州的天气"}))
4print(f"ReActAgent 最终输出结果：{result.get("output")}")

完整示例代码

PYTHON
1import asyncio
2import os
3from datetime import datetime
4

5from openjiuwen.agent.react_agent import create_react_agent_config, ReActAgent
6from openjiuwen.core.component.common.configs.model_config import ModelConfig
7from openjiuwen.core.utils.llm.base import BaseModelInfo
8from openjiuwen.core.utils.tool.param import Param
9from openjiuwen.core.utils.tool.service_api.restful_api import RestfulApi
10

11API_BASE = os.getenv("API_BASE", "your api base")
12API_KEY = os.getenv("API_KEY", "your api key")
13MODEL_NAME = os.getenv("MODEL_NAME", "")
14MODEL_PROVIDER = os.getenv("MODEL_PROVIDER", "")
15os.environ.setdefault("LLM_SSL_VERIFY", "false")  # 关闭SSL校验仅用于本地调试，生产环境请务必打开
16os.environ.setdefault("RESTFUL_SSL_VERIFY", "false")  # 关闭SSL校验仅用于本地调试，生产环境请务必打开
17

18def build_current_date():
19    current_datetime = datetime.now()
20    return current_datetime.strftime("%Y-%m-%d")
21

22class ReactAgentImpl:
23    @staticmethod
24    def _create_model():
25        return ModelConfig(model_provider=MODEL_PROVIDER,
26                           model_info=BaseModelInfo(
27                               model=MODEL_NAME,
28                               api_base=API_BASE,
29                               api_key=API_KEY,
30                               temperature=0.7,
31                               top_p=0.9,
32                               timeout=30  # 添加超时设置
33                           ))
34

35    @staticmethod
36    def _create_tool():
37        weather_plugin = RestfulApi(
38            name="WeatherReporter",
39            description="天气查询插件",
40            params=[
41                Param(name="location", description="天气查询的地点，必须为英文", type="string", required=True),
42                Param(name="date", description="天气查询的时间，格式为YYYY-MM-DD", type="string", required=True),
43            ],
44            path="user's path to weather service",
45            headers={},
46            method="GET",
47            response=[],
48        )
49        return weather_plugin
50

51    @staticmethod
52    def _create_prompt_template():
53        system_prompt = "你是一个AI助手，在适当的时候调用合适的工具，帮助我完成任务！今天的日期为：{}\n注意：1. 如果用户请求中未指定具体时间，则默认为今天。"
54        return [
55            dict(role="system", content=system_prompt.format(build_current_date()))
56        ]
57

58async def main():
59    model_config = ReactAgentImpl._create_model()
60    prompt_template = ReactAgentImpl._create_prompt_template()
61

62    react_agent_config = create_react_agent_config(
63        agent_id="react_agent_123",
64        agent_version="0.0.1",
65        description="AI助手",
66        model=model_config,
67        prompt_template=prompt_template
68    )
69

70    react_agent: ReActAgent = ReActAgent(react_agent_config)
71    react_agent.add_tools([ReactAgentImpl._create_tool()])
72

73    result = await react_agent.invoke({"query": "查询杭州的天气"})
74    print(f"ReActAgent 最终输出结果：{result}")
75

76if __name__ == "__main__":
77    asyncio.run(main())
